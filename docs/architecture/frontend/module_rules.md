# Правила модулей frontend

## Модули

Frontend-модуль - папка внутри `src/api`, `src/components`, `src/features`, `src/pages`, `src/domain` или `src/utils` с четкой ответственностью.

## Правила

- API-запросы размещаются в `src/api`.
- UI-компоненты не выполняют прямые запросы, если для сценария уже есть feature hook или API-модуль.
- Feature-модуль владеет своим context, hooks и состоянием сценария.
- `src/domain` содержит только frontend-типы, конфигурацию и чистые утилиты.
- Страницы собирают готовые компоненты и сценарии.
- Frontend не дублирует backend-бизнес-логику.

## Публичная поверхность

Публичная поверхность модуля должна быть минимальной. Если модуль имеет `index.js`, внешний код импортирует модуль через него.

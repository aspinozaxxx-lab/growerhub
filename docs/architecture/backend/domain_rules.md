# Правила доменов backend

## Структура домена

Домен размещается в `ru.growerhub.backend.<domain>`.

Обязательная структура:

- `<Domain>Facade.java` в корне доменного пакета.
- `contract` - публичные DTO, records, enums, interfaces и публичные исключения домена.
- `engine` - бизнес-логика и use case сервисы.
- `jpa` - entity и repository домена, если домен владеет данными в БД.

REST-контроллеры, MQTT-обработчики, фильтры, конфигурация, инициализаторы, ресурсы и внешние шлюзы не размещаются внутри доменного пакета, если они не являются частью бизнес-ядра домена.

## Публичная поверхность

Facade является единственной точкой входа в домен. Публичные операции Facade образуют публичный API домена.

Внешний код может импортировать только Facade домена и типы из `contract`. Импорт `engine` и `jpa` другого домена запрещен.

## Адаптеры

Один REST-обработчик или MQTT-потребитель вызывает одну публичную операцию Facade. Если нужен сценарий из нескольких операций, он оформляется как композитная операция в Facade домена-владельца сценария.

Адаптер содержит только разбор входа, проверку технических параметров, вызов Facade и маппинг ответа. Бизнес-решения в адаптере запрещены.

## Транзакции

Транзакционные границы задаются на публичных операциях Facade. `engine` и `jpa` работают внутри транзакции, открытой Facade.

## Конфигурация

Параметры доменов и подсистем задаются через `application.yml` и читаются через классы настроек. Таймауты, интервалы, лимиты, пороги, дефолтные значения, флаги поведения, URL, пути и топики не задаются в доменах как константы.

## Документ домена

Каждый домен имеет документ `docs/architecture/backend/domains/<domain>.md` только со следующими разделами:

- Назначение.
- Публичный Facade.
- Публичные контракты.
- Владение данными.
- Используемые домены.
- Внешние пользователи домена.
- Алгоритм работы.
- Ограничения.

Лимиты:

- Назначение - максимум 500 символов.
- Владение данными - максимум 700 символов.
- Алгоритм работы - максимум 1000 символов.
- Ограничения - максимум 700 символов.

Если текст превышает лимит, он ужимается до summary.

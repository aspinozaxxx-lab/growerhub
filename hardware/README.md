# GrowerHub Controller — Hardware (v0.1)

Эта папка фиксирует аппаратную архитектуру, пинауты и соединения устройства управления поливом.

Назначение документов:
- служат «источником правды» по железу
- используются перед переносом схемы в KiCad
- синхронизируются с firmware (GPIO и интерфейсы)

Состав устройства:
- Питание: 12V DC
- DC-DC: отдельный модуль 12V → 5V
- MCU: ESP32 devboard
- Насос: 12V DC
- Датчики влажности почвы: 2× RJ9 (аналоговые)
- RTC модуль: I2C
- Опционально: DHT22

Файлы:
- power.md — питание и архитектура линий питания
- connectors.md — распиновка разъёмов
- mcu.md — используемые GPIO (по firmware)
- rtc.md — подключение RTC
- noise-notes.md — помехи и работа АЦП

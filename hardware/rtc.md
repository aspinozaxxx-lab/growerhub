# RTC module

RTC модуль используется для хранения реального времени при отсутствии сети.

## Интерфейс
- Тип: I2C
- SDA: RTC_SDA (см. mcu.md)
- SCL: RTC_SCL (см. mcu.md)

## Питание
- VCC: 3.3V
- GND: GND
- Backup battery: присутствует / TODO

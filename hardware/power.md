# Power architecture

## Вход питания
- VIN: 12V DC
- Один внешний блок питания

## DC-DC
- Отдельный модуль: 12V → 5V
- 5V используется как промежуточная шина

## Напряжения в системе
| Rail | Назначение |
|----|-----------|
| 12V | Насос |
| 5V  | Вход devboard ESP32 |
| 3.3V | ESP32 + датчики + RTC |

3.3V формируется стабилизатором на ESP32 devboard.

## Разделение питания насоса и логики
Несмотря на один блок питания:
- насос питается напрямую от 12V
- логика питается от 12V → DC-DC → 5V → 3.3V

Рекомендации:
- линии 12V на насос и на DC-DC разводить отдельно
- возврат тока насоса не должен идти через землю логики
- точка объединения GND — у входа питания (звезда)

## Конденсаторы (рекомендации)
- На выходе DC-DC (5V): 470–1000 µF
- На 3.3V возле ESP32: 10–100 µF + 0.1 µF
- Локально у разъёмов датчиков: 0.1 µF (опционально)
